Blog Application

A simple yet powerful Django-based web application that allows users to create blogs and publish posts under each blog.

Features

Create, edit, and delete Blogs

Create, edit, and delete Posts under each blog

User authentication (login, logout, register)

Clean, responsive UI (Bootstrap recommended)

Admin panel for managing blogs and posts

Slug-based URLs for SEO-friendly links

Django ORM-powered models

Extensible architecture for comments, tags, or categories
